# План развития Qwen Medical Transcriber

## Обзор проекта

**Qwen Medical Transcriber** — профессиональная система офлайн транскрипции медицинских консультаций с разделением диалога по ролям.

**Текущий стек:**
- Backend: FastAPI/Gradio (Web UI), processor.py (orкестратор)
- ASR: Qwen3-ASR-1.7B (Hugging Face)
- Diarization: pyannote.audio 3.1
- Aligner: Qwen3-ForcedAligner-0.6B
- UI: Gradio (web), CustomTkinter (Win32)
- Output: Word (.docx)

## Текущие проблемы (из README)

- Модели не включены в репозиторий (требуют загрузки вручную)
- GPU поддержка требует ручной установки CUDA
- Медленная обработка для длинных аудио (>15 мин)
